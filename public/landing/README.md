# Obrázky pre úvodnú stránku

Sem dáš obrázky, na ktoré odkazuje `src/site/siteConfig.js`. Všetky sú **voliteľné** —
kým ich nedoplníš, zobrazí sa elegantná farebná výplň.

- `hero.jpg` — veľký úvodný obrázok (vizualizácia budovy). Odporúčané ~1920×1080.
- `byt-1.jpg`, `byt-2.jpg`, `byt-3.jpg` — náhľady na karty bytov. Odporúčané ~800×550.

Názvy musia sedieť s `image:` v `siteConfig.js` (alebo si tam cesty uprav).
